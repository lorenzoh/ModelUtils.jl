ModelUtils.jl documentation

ModelUtils.jl provides utilities for working with Flux.jl models.

Features

  • initialization
  • hooks
  • iteration

The most important type it defines is Model, a tree data structure that wraps your Flux model and is itself a Flux layer.

ModelUtils.jl should work with all models that work with Flux, but for some custom types you might have to extend a few of ModelUtils.jls methods, see Custom models for more information.
